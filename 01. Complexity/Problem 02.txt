What is the expected running time of the following C# code? Explain why.
Assume the input matrix has size of n * m.

long CalcCount(int[,] matrix)
{
    long count = 0;
    for (int row=0; row<matrix.GetLength(0); row++)
        if (matrix[row, 0] % 2 == 0)
            for (int col=0; col<matrix.GetLength(1); col++)
                if (matrix[row,col] > 0)
                    count++;
    return count;
}

SOLUTION: В най-лощия случай сложността е О(n*m). Ако допуснем, че не винаги ще имаме 0-еви ст-йности, тогва може да кажем, че сложността може да е  О(n*m/2)
